Automation · Google Apps Script

Mapogo Agent Leaderboard

A live leaderboard system built in Google Apps Script that automatically ranks agents based on daily enrollment data. Originally built to replace a manual spreadsheet updated by hand every morning.

What it does
  • Event-driven triggers fire on every data change — no manual refresh
  • Top 3 standings auto-populate with agent photos, names, and policy counts
  • Daily data entry table tracks MAPDs, AORs, LIFE, and HIP across all agents
  • Movement tracker shows position changes day-over-day with arrow indicators
  • Full history table with manual input support for corrections
Why it matters
  • Demonstrates trigger-based automation, conditional logic, and data binding
  • Real tool used by a real team — not a coding exercise
  • Shows how a small script can replace a recurring manual process
2D Pixel RPG · C# · Unity · Capstone Project

Ethical Odyssey

A fully playable 2D pixel RPG developed as my Senior High School capstone at STI Tagum College. Built in C# with Unity and Ink for dialogue. The core mechanic is a branching decision system where every choice is classified as ethical, neutral, or non-ethical.

What I built
  • Branching dialogue system with ethical / neutral / non-ethical decision tree using Ink language
  • Player state tracking that persists choices across the entire playthrough
  • Story routing logic that changes outcomes based on cumulative decisions
  • Custom pixel art and tile-based level design
  • Full C# codebase built in Unity
Why it matters
  • Designing branching logic taught me to map decision trees before coding — a habit that carries directly into scripting and automation
  • State management across long playthroughs gave me real practice with conditional flows
  • Successfully defended as my SHS capstone project
